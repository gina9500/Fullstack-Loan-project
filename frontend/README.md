###### 贷款风险评估系统前端项目（chenjinjin 20251117~20251219）######

## 1.项目简介

基于 React 框架开发，提供个人贷款(本次只跳转显示)和企业贷款两种申请类型。
系统支持用户登录、企业贷款信息填写、财务数据文件上传、信息确认以及贷款风险评估结果等功能。

## 2.技术栈

- **前端框架**: React
- **路由管理**: React Router
- **构建工具**: Vite

## 3.React 项目结构

src/
├── api/ # API 请求接口
├── components/ # 可复用组件
│ ├── form/ # 表单相关组件
│ └── layout/ # 布局组件
├── pages/ # 页面组件
├── routes/ # 路由配置
├── utils/ # 工具函数
├── config/ # 配置文件
└── assets/ # 静态资源

## 4.核心业务功能

- 认证系统
  登录页面实现用户身份验证
  根据用户名判断用户类型（个人/企业）并跳转到对应入力页面

- 贷款申请流程
  个人入力：个人（暂只做跳转，不做实现）
  企业入力：提供企业用户的贷款申请入力表单，包含企业基本信息、贷款信息等
  贷款信息确认：已提交的贷款信息，供用户确认
  贷款风险评估结果：结果显示

- 表单处理功能
  统一的表单验证逻辑
  错误处理和提示
  财务文件上传功能
  动态下拉框

- 组件化设计
  基础表单组件：InputField、SelectField、FileUploadField
  布局组件：BaseLayout 提供统一的页面布局
  页面组件：每个功能页面独立封装

- 状态管理
  使用 React Hooks (useState)进行本地状态管理
  表单数据和错误信息分别管理

- API 交互（未实现）
  封装了 request 工具函数处理 HTTP 请求
  提供了用户相关和贷款相关的 API 接口

- 路由配置
  使用 React Router 进行页面路由管理

- 数据流与业务逻辑
  用户登录 → 身份验证 → 根据用户类型跳转
  填写贷款入力表单 → 表单验证 → 提交数据 → 存储
  查看确认页面 → 确认信息 → 提交确认 → 显示结果页面
